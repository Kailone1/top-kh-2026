**Мета роботи** - метою роботи є вивчення синтаксису функцій, опанування механізмів передачі аргументів та методів виклику функцій. Окрему увагу приділено концепції передачі функцій як параметрів (callback), що є важливим інструментом для побудови гнучких алгоритмів у процедурному програмуванні.

## Хід роботи

1. Ми розділили код на дві окремі частини. Перша функція просто видає список із готовими фразами. Друга функція бере кожну з цих фраз по черзі і підставляє її у заздалегідь підготовлений шаблон речення. Це дозволяє зробити програму акуратнішою, бо ми окремо зберігаємо дані, а окремо — логіку їхнього оформлення.

Код:
```python
def list_benefits():
    return "More organized code", "More readable code", "Easier code reuse", "Allowing programmers to share and connect code together"

def build_sentence(benefit):
    return "%s is a benefit of functions!" % benefit

def name_the_benefits_of_functions():
    list_of_benefits = list_benefits()
    for benefit in list_of_benefits:
        print(build_sentence(benefit))

name_the_benefits_of_functions()
```
