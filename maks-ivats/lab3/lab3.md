## Хід роботи

Код послідовно перевіряє умови, і оскільки всі вони істинні, виведеться кожен рядок.Ось короткий опис того, що перевіряється:
1. Чи число 20 більше 15 (True).
2. Чи масив first_array не порожній (True).
3. Чи довжина second_array дорівнює 2 (True).
4. Чи сума довжин обох масивів ($3 + 2$) дорівнює 5 (True).
5. Чи масив існує і чи його перший елемент — 1 (True).
6. Чи змінна second_number має значення None або False (True).

Код:
```python
number = 20
second_number = None
first_array = [1,2,3]
second_array = [1,2]

if number > 15:
    print("1")

if first_array:
    print("2")

if len(second_array) == 2:
    print("3")

if len(first_array) + len(second_array) == 5:
    print("4")

if first_array and first_array[0] == 1:
    print("5")

if not second_number:
    print("6")
```

2. Ми освоюємо роботу циклу for на прикладі вкладених циклів.

    Цей код демонструє, як створювати всі можливі комбінації з двох списків. Програма бере один елемент із першого списку і "проганяє" крізь нього весь другий список, повторюючи це для кожного прикметника. (Завдання з сайту w3school)

Код:
```python
adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

for x in adj:
  for y in fruits:
    print(x, y)

```

3. Ми освоюємо роботу циклу while, який виконує дії доти, доки справджується певна умова. (Завдання з сайту w3school)

Код:
```python
i = 1
while i < 6:
  print(i)
  i += 1
else:
  print("i is no longer less than 6")
```
