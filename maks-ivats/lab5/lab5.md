**Мета роботи** - опанувати навички створення класів та об'єктів у мові програмування Python 3. Навчитися визначати атрибути класу, реалізовувати методи та створювати обчислювані властивості об'єктів для автоматизації обробки даних.


## Хід роботи [lab5.py](lab5.py)

1. Створення класу: Оголосив клас Student для опису структури даних студента.
2. Налаштування атрибутів: Через конструктор __init__ задав властивості (ім'я та оцінки) та реалізував автоматичний розрахунок середнього бала.
3. Розробка методу: Написав метод get_report для генерації текстового опису об’єкта.
4. Створення об'єктів: Створив кілька екземплярів класу з різними даними.
5. Тестування: Викликав методи через створені об'єкти для перевірки коректності роботи коду.

## Висновок
Під час виконання роботи було вивчено базові концепції об'єктно-орієнтованого програмування. Я навчився групувати дані та функції в одну логічну одиницю — клас.

Найбільш практичним аспектом виявилася можливість створення динамічних атрибутів (як-от середній бал), які розраховуються автоматично. Це дозволяє уникнути дублювання коду та забезпечує цілісність даних: кожен об'єкт самостійно керує своїм станом та надає готовий результат через власні методи.
